## 기본 사항

폴더명 `{identifier}_{language}`

이 뒤에 오는 것들은 작성 언어입니다. 다음은 현재까지 작성된 언어의 예시.

작성언어 | 폴더 뒤 | 확장자
:---:|:---:|:---:
 C | C | .c
 C++ | cpp | .cpp
 Python 또는 pypy | python | .py
 Kotlin | kotlin | .kt
 Lua | lua | .lua
