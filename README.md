## 기본 사항

폴더명 `{identifier}_{language}`

`identifier`의 종류는 다음과 같음.

* `zeta`는 [BOJ](https://www.acmicpc.net/)입니다.

`language`는 작성 언어로 현재까지 작성된 언어 및 예시는 다음과 같음.

작성언어 | 폴더 뒤 | 확장자
:---:|:---:|:---:
 C | C | .c
 C++ | cpp | .cpp
 Python 또는 pypy | python | .py
 Kotlin | kotlin | .kt
 Lua | lua | .lua

std
* C++17
* C11
